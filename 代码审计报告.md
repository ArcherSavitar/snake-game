## 2. 清理内容
### 2.1 无效代码移除
- `start_game.py`
- 移除未使用导入: `sys`、`random`
- 将裸 `except` 改为 `except OSError`
- 清理无意义 `f-string`
- 使用 `ReusableTCPServer` 与上下文管理器，避免资源泄漏

- `test_server.py`
- 改为断言式自动化测试，移除调试输出依赖
- 增加 `shutdown + join`，修复线程关闭异常

- `validate_readme.py`
- 移除无效检查分支（仅 `pass` 的表格检查逻辑）
- 收敛为明确规则，减少冗余输出

- `index.html`
- 删除注释垃圾与重复输入分支
- 统一输入处理函数，降低重复实现
- 游戏循环改为 `setTimeout` 调度，速度变化即时生效

- `MainActivity.java`
- 增加 `onDestroy` 释放 `WebView`

### 2.2 重复实现处理
- 将 Android assets 中的 `index.html` 与根目录 `index.html` 同步，消除双份逻辑漂移风险。
